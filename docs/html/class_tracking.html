Start or restart a tracking process with an new tracker.

Initializes a new tracker with the given parameters and sets the tracker's state.

Parameters
first_fThe initial frame used for tracker initialization.
roiBounding box of the region of interest to track.
_stateInitial state assigned to the tracker. Default value is TCR_RUNN.
hogWhether to use HOG features. Default value is true.
fixed_windowWhether to use a fixed window size. Default value is true.
multiscaleWhether to enable multi-scale tracking. Default value is true.
labWhether to include Lab color features. Default value is true.
Returns
Boolean value. Return true if the initialization goes on properly.
